## How to Access Sales Entry

1. Open the Sales Dashboard in your web browser
2. Login with your username and password
3. Click on **"Sales Entry"** in the left sidebar menu

---

## Warehouse Sale Entry (Category-Based Sales)

### Step 1: Select the Warehouse Sale Tab

- Click on the **"Warehouse Sale"** tab at the top of the sales entry form
- This tab has a warehouse icon üì¶

### Step 2: Fill in Basic Information

#### Select Date
- Click on the **Date** field
- Choose the date for which you are entering sales
- Default is today's date

#### Select Branch
- Click on the **Branch** dropdown
- Select your branch (e.g., "D WATSON PWD", "D WATSON G 15", etc.)
- ‚ö†Ô∏è **IMPORTANT:** You can only select branches you have access to

### Step 3: Enter Sales by Category

- You will see input fields for each category (e.g., COSTMAIES, MEDICINE, etc.)
- For each category, enter:
  - **Sales Amount:** Enter the total sales amount for that category
  - **Cost Amount:** Enter the total cost for that category
- Leave fields as **0 (zero)** if there are no sales for that category

### Step 4: Check Totals

The form automatically calculates:
- **Total Sales** (sum of all category sales)
- **Total Cost** (sum of all category costs)
- **Total Profit** (Total Sales - Total Cost)

### Step 5: Add Notes (Optional)

- You can add any notes or comments in the **Notes** field
- This is optional but helpful for reference

### Step 6: Save the Sale

- Click the **"Save Sale"** button
- You will see a success message
- The sales list will automatically open showing your entry

## Shop Sale Entry (Department/Sub-Department Sales)

### Step 1: Select the Shop Sale Tab

- Click on the **"Shop Sale"** tab at the top of the sales entry form
- This tab has a shop icon üè™

### Step 2: Fill in Basic Information

#### Select Date
- Click on the **Date** field
- Choose the date for which you are entering sales
- Default is today's date

#### Select Branch
- Click on the **Branch** dropdown
- Select your branch
- ‚ö†Ô∏è **IMPORTANT:** You can only select branches you have access to

#### Select Department
- After selecting branch, the **Department** dropdown will load
- Select the department (e.g., "COSMATICES", etc.)

### Step 3: Enter Sub-Department Sales

- After selecting department, sub-departments will appear in a table
- For each sub-department, enter:
  - **Gross Sale:** Enter the total gross sale amount
  - **Discount:** Enter discount amount (if any)
  - **Return:** Enter return amount (if any)
  - **GST:** Enter GST amount (if any)

The system automatically calculates:
- **Sale Value** = Gross Sale - Discount
- **Net Sale** = Sale Value - Return + GST

### Step 4: Save the Sale

- Click the **"Save Bulk Department Sales"** button
- You will see a success message
- The department sales list will automatically open

## Viewing Sales List

### To View Sales You Entered

1. After saving, the sales list opens automatically
2. Or click **"View Sales List"** button in the Warehouse Sale tab
3. Or click **"View Department Sales List"** button in the Shop Sale tab

### Filtering Sales

- Select **"From Date"** and **"To Date"** to filter by date range
- Select **Branch** to filter by specific branch
- Select **Category/Department** to filter by category or department
- Click **"Apply Filter"** to see filtered results

### View Options

- **Card View:** Shows sales as cards (default)
- **Table View:** Shows sales in a table format
- Click the buttons to switch between views

## Deleting a Sale

### To Delete a Sale

2. Find the sale you want to delete

4. Confirm the deletion when prompted

## Common Issues & Solutions

| Issue | Solution |
|-------|----------|
| "Please select a branch before saving" | Make sure you have selected a branch from the dropdown |
| "Please select a date before saving" | Make sure you have selected a date |
| Branch dropdown is empty | Contact your administrator - you may not have access to any branches |
| Department dropdown is empty | Make sure you have selected a branch first, then the department will load |
| Sub-departments not showing | Make sure you have selected both branch and department first |
| Cannot edit/delete a sale | You may not have permission. Contact your administrator |
| Numbers not showing properly when zoomed out | This has been fixed. Refresh the page if you still see issues |
